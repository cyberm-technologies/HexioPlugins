# HexioPlugins

This repository contains all the resources needed to build the free, open-source compiler plugins for the [Hexio C2 Framework](https://hexioframework.com).

Each plugin packages a platform-specific compiler toolchain into a Docker container that can be uploaded to a Hexio Teamserver. See the individual plugin READMEs for build and installation instructions.
